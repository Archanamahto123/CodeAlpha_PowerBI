# 📊 Financial Health Dashboard

## 📌 Project Overview

This project is developed as part of the CodeAlpha Power BI Internship.
The dashboard provides a comprehensive analysis of an organization's
financial performance using interactive Power BI visualizations.

## 🎯 Objectives

-   Analyze financial performance.
-   Track income, expenses, and profit.
-   Monitor cash flow trends.
-   Support budgeting and financial planning.
-   Provide interactive insights using Power BI.

## 🛠️ Tools Used

-   Power BI Desktop
-   Microsoft Excel
-   Power Query
-   DAX

## 📈 Dashboard Features

-   KPI Cards
-   Revenue & Expense Trend Analysis
-   Profit Analysis
-   Cash Flow Overview
-   Forecasting
-   Interactive Slicers
-   Drill Down
-   Custom Tooltips
